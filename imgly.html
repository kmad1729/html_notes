img tag

img tag is used in html to display images.

Images, like links can be be locally linked i.e. from a local source or can be an absolute link (like here). Use src attribute to link images. Also specify height and width attributes. This is helpful when a browser renders an html page, if height and width attributes are not specified, it calculates the values while loading the image. If specified, the browser calculates the size when the page loads. This avoids the image jumping around when a page loads.

No closing tag is required for img as it does not enclose any content.
